**20/02 - Aula 01: JavaScript: Sintaxe, tipos de dados, variáveis let e const, escopo e operadores.**

### 1. Sintaxe e Variáveis (`let` e `const`)

O JavaScript moderno utiliza formas específicas para declarar variáveis, abandonando o antigo `var` para evitar bugs de escopo.

* **let:** Permite criar variáveis cujo valor pode ser alterado (mutáveis).
* *Exemplo:* `let idade = 25; idade = 26;`


* **const:** Cria constantes cujo valor não pode ser reatribuído após a declaração.
* *Exemplo:* `const pi = 3.1415;`



### 2. Tipos de Dados

O JS é uma linguagem de tipagem dinâmica. Os principais tipos básicos são:

* **Number:** Números inteiros ou decimais (ex: `10`, `15.5`).
* **String:** Textos entre aspas (ex: `"Olá Mundo"`).
* **Boolean:** Valores lógicos (`true` ou `false`).
* **Null e Undefined:** Representam ausência de valor ou variável não inicializada.
* **Object e Array:** Estruturas para conjuntos de dados.

### 3. Escopo

Define onde uma variável é acessível dentro do código:

* **Escopo de Bloco:** Variáveis declaradas com `let` e `const` dentro de `{ }` (como em um `if` ou `for`) só existem ali dentro.
* **Escopo Global:** Variáveis declaradas fora de qualquer função ou bloco, acessíveis em todo o script.

### 4. Operadores

* **Aritméticos:** `+`, `-`, `*`, `/`, `%` (resto), `**` (potência).
* **Atribuição:** `=`, `+=`, `-=`.
* **Comparação:** `==` (valor), `===` (valor e tipo estrito), `!=`, `!==`.
* **Lógicos:** `&&` (E), `||` (OU), `!` (NÃO).
