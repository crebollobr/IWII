
**Aula 02: JavaScript: Estruturas de controle de fluxo e laços de repetição.**

### 1. Estruturas Condicionais (Controle de Fluxo)

Servem para o código tomar decisões baseadas em condições booleanas.

* **if / else if / else:** Executa blocos de código diferentes dependendo da condição.
* *Exemplo:*
```javascript
let hora = 14;
if (hora < 12) {
    console.log("Bom dia");
} else if (hora < 18) {
    console.log("Boa tarde");
} else {
    console.log("Boa noite");
}

```




* **switch:** Ideal para verificar múltiplas condições fixas para uma mesma variável.
* *Exemplo:*
```javascript
let cor = "azul";
switch (cor) {
    case "azul": console.log("Cor do céu"); break;
    case "verde": console.log("Cor da mata"); break;
    default: console.log("Cor não identificada");
}

```





### 2. Laços de Repetição (Loops)

Utilizados para executar o mesmo bloco de código várias vezes.

* **for:** Usado quando sabemos exatamente quantas vezes o código deve ser repetido.
* *Exemplo:*
```javascript
for (let i = 0; i < 5; i++) {
    console.log("Repetição nº: " + i);
}

```




* **while:** Executa enquanto uma condição for verdadeira (útil quando não sabemos o número exato de repetições).
* *Exemplo:*
```javascript
let contador = 0;
while (contador < 3) {
    console.log("Contando...");
    contador++;
}

```




* **do...while:** Semelhante ao while, mas garante que o código seja executado **pelo menos uma vez** antes de testar a condição.

### 3. Controle de Loops

* **break:** Interrompe o laço imediatamente.
* **continue:** Pula a iteração atual e vai para a próxima.

---

**Exemplo de Prática Combinada:**

```javascript
// Percorrendo números e identificando pares
for (let n = 1; n <= 10; n++) {
    if (n % 2 === 0) {
        console.log(n + " é par");
    } else {
        console.log(n + " é ímpar");
    }
}

```
